# Hook App - React + TypeScript

Esta aplicación es un proyecto de práctica extraído del curso "React: De cero a experto (Hooks y MERN)" impartido por Fernando Herrera en Udemy ([@Klerith](https://www.udemy.com/course/react-cero-experto/)).

## Descripción

Hook App es una aplicación de demostración que ilustra el uso de los diferentes Hooks en React, implementada con TypeScript. La aplicación incluye ejemplos prácticos de:

- useState
- useEffect
- useRef
- useLayoutEffect
- useReducer
- Custom Hooks

## Características Principales

- Implementación de formularios controlados
- Manejo de estados complejos
- Consumo de APIs (Pokemon API)
- Custom Hooks reutilizables
- Implementación de contador con diferentes variantes
- Aplicación de TODO con useReducer
- Estilos con Bootstrap 5

## Requisitos Previos

- Node.js (versión 14 o superior)
- npm o yarn

## Instalación

1. Clona el repositorio:
```bash
git clone <url-del-repositorio>
```

2. Instala las dependencias:
```bash
npm install
# o
yarn install
```

## Scripts Disponibles

- `npm run dev`: Inicia el servidor de desarrollo
- `npm run build`: Genera la versión de producción
- `npm run lint`: Ejecuta el linter para verificar el código
- `npm run preview`: Previsualiza la versión de producción

## Tecnologías Utilizadas

- React 18
- TypeScript
- Vite
- Bootstrap 5
- ESLint

## Estructura del Proyecto

El proyecto está organizado en diferentes carpetas que muestran ejemplos de implementación de Hooks:

- `/src/01-useState`: Ejemplos de uso del Hook useState
- `/src/02-useEffect`: Implementaciones con useEffect
- `/src/03-example`: Ejemplos de uso múltiple de Custom Hooks
- `/src/04-useRef`: Ejemplos con useRef
- `/src/05-useLayoutEffect`: Implementaciones con useLayoutEffect
- `/src/06-useReducer`: Implementaciones con useReducer y TODO App
- `/src/hooks`: Custom Hooks reutilizables
